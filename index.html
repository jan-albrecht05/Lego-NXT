LEGO NXT Basisset

Wer von uns hat als Kind nicht von einem eigenen Roboter geträumt?
Mit diesem LEGO Set kann dieser Wunsch für jeden Wirklichkeit werden. Die Teile sind für zahlreiche Modelle ausreichend und durch andere Sets erweiterbar. Mit einem "Drag-and-Drop" Programm kann man die gebauten Roboter zum Leben erwecken - selbst, wenn du noch nie eine Zeile Code geschrieben hast!
